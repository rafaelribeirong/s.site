Lab Lion - Detalhes do Projeto
Resumo do Projeto

Exploração completa de um ambiente vulnerável contendo:

  • SQL Injection levando a RCE
  • Upload de webshell maliciosa
  • Escalonamento de privilégio via Cron Jobs
Metodologia
  1. Reconhecimento: Varredura com Nmap
  2. Exploração: SQLi → RCE via INTO OUTFILE
  3. Pós-exploração: Webshell e shell reverso
  4. Privilege Escalation: Exploração de Cron Job
Técnicas Utilizadas
SQL Injection RCE Webshell Cron Job Hijacking LinPEAS
