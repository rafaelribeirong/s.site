Análise de Vulnerabilidades em Aplicação Web
Resumo do Projeto

Análise detalhada de vulnerabilidades em uma aplicação web vulnerável (DVWA), incluindo:

  • SQL Injection
  • Cross-Site Scripting (XSS)
  • Cross-Site Request Forgery (CSRF)
  • Command Injection
Metodologia

Utilização de Burp Suite para interceptação e manipulação de requisições, exploração manual de vulnerabilidades e desenvolvimento de scripts para automação de testes.

Resultados

Identificação de 8 vulnerabilidades críticas, com recomendações de mitigação para cada uma.
